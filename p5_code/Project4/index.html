Health Catch

(Interactive Game If I can get it to freaking work) This is a health food interface game I made. I added a pixel art interface, along with heath food facts about certain food items. I used p5play, sound, and scenemanager libraries to create this short time consuming mini game. The concept of the game is the food item is colliding with a flat board while it has a sound effect along with the the food thats hitting the board.
