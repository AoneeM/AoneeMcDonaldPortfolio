Grid Pattern

For this excerise I combined the grid code with text, and color changing of a shape. When I was making this I did struggle adding codes together to make it work, but I did end up with a little small something.

The canvas changes color everytime the grid finishes its cycle (while also changing the pattern of when grid starts.) When the mouse is pressed, the circle in the middle changes and a mesage shows up on the screen. However, when the grid changes the cicrle with the text goes back to the cyan colored circle.
